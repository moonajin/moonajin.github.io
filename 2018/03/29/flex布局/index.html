Flex弹性布局
瑾瑾 前端开发工程师
 2018-03-29  
  • css 

一、基本概念

  • 采用 flex 布局的元素，我们称为flex 容器
    • 容器默认存在两根轴：主轴和垂直交叉轴
  • 该元素的所有子元素自动成为容器中的成员，称为 flex 项目
    • 项目默认沿着主轴排列

二、容器属性

1、flex-direction

  • flex-direction属性决定主轴的方向（即项目的排列方向）
    • row（默认）：主轴为水平方向，从左向右排列
    • row-reverse：主轴为水平方向，从右向左排列
    • column：主轴为垂直方向，从上往下排列
    • column-reverse：主轴为垂直方向，从下往上排列

2、flex-wrap

  • flex-wrap属性决定换行方式：是否换行、如何换行

  • 默认情况下，项目都是排在一条轴线上的，但是如果一条轴线排不下这些项目，就会涉及到换行的问题

    • nowrap（默认）：不换行

    • wrap：换行，从第一行依次向下排列，第一行在上方

    • wrap-reverse：换行，从第一行依次向上排列，第一行在下方

3、justify-content

  • justify-content属性决定项目在主轴上的对齐方式

    • flex-start（默认）：左对齐

    • flex-end：右对齐

    • center：居中对齐

    • space-between：两端对齐，项目之间的间隔都相等

    • space-around：每个项目两侧的间隔相等，项目之间的间隔比项目与边框的间隔大一倍

4、align-items

  • align-items属性决定了项目在交叉轴上的对齐方式
    • stretch（默认）：如果项目未设置高度或设为auto，将占满整个容器的高度
    • flex-start：交叉轴的起点对齐
    • flex-end：交叉轴的终点对齐
    • center：交叉轴的中间点对齐
    • baseline: 项目的第一行文字的基线对齐

5、align-content

  • align-content属性定义了多根轴线的对齐方式
  • 如果项目只有一根轴线，该属性则不起作用
    • stretch（默认）：轴线占满整个交叉轴
    • flex-start：与交叉轴的起点对齐
    • flex-end：与交叉轴的终点对齐
    • center：与交叉轴的中间点对齐
    • space-between：与交叉轴两端对齐，轴线之间的间隔平均分布
    • space-around：每根轴线两侧的间隔都相等，轴线之间的间隔比轴线与边框的间隔大一倍

6、flex-flow

  • flex-flow属性是flex-direction和flex-wrap属性的简写形式
  • 默认值为row nowrap

三、项目属性

1、align-self

  • align-self属性允许单个项目有与其他项目不一样的对齐方式，可覆盖 align-items 属性
  • 默认值为auto，表示继承父元素的align-items属性
  • 如果没有父元素，则等同于stretch
    • auto（默认）：继承父元素的align-items属性
    • stretch：如果项目未设置高度或设为auto，将占满整个容器的高度
    • flex-start：交叉轴的起点对齐
    • flex-end：交叉轴的终点对齐
    • center：交叉轴的中间点对齐
    • baseline: 项目的第一行文字的基线对齐

2、order

  • order属性定义了项目的排列顺序

  • 数值越小，顺序越靠前，默认为0

3、flex-grow

  • flex-grow属性定义了项目的放大比例

  • 默认为0，即如果存在剩余空间，也不放大

  • 如果所有项目的flex-grow属性都为1且存在剩余空间，那么它们将等分剩余空间

  • 如果一个项目的flex-grow属性为2，其他项目都为1，那么该项目占据的剩余空间将比其他项多一倍

4、flex-shrink

  • flex-shrink属性定义了项目的缩小比例

  • 默认为1，即如果空间不足，该项目将缩小

  • 如果所有项目的flex-shrink属性都为1，当空间不足时，那么它们将等比例缩小

  • 如果一个项目的flex-shrink属性为0，其他项目都为1，那么当空间不足时，该项目不缩小，其他项目将等比例缩小

  • 负值对该属性无效

5、flex-basis

  • flex-basis属性定义了在分配剩余空间之前，项目占据的主轴空间
  • 浏览器根据这个属性，计算主轴是否有多余空间
  • 默认为auto，即项目本身的大小
  • 该属性可以设为跟width或height属性一样的值，如果该属性值为100px，那么该项目将占据固定空间 100px

6、flex

  • flex属性是flex-grow，flex-shrink和flex-basis的简写
  • 默认值为0 1 auto，即不放大会缩小
  • 该属性有两个快捷值：
    • auto：等价于1 1 auto，即放大且缩小
    • none：等价于0 0 auto，即不放大也不缩小
  • flex：1
    • 等价于1 1 0%，经常用作自适应布局，设置为flex：1的项目会自动放大占满剩余空间

四、其他

  • 注意：容器设为 Flex 布局后，子元素的float、clear和vertical-align属性将会失效
