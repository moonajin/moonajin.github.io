VUE中将px转成rem单位
瑾瑾 前端开发工程师
 2021-10-15  
  • VUE 

在开发移动端项目时，需要对不同尺寸的设备进行适配，页面中字体、按钮等元素的大小与屏幕大小（屏幕的宽度）有关，可以通过 rem 来进行适配。

rem 适配

  • rem 是 CSS3 新增的一个相对单位，相对的是 HTML 根元素上的 font-size 的大小。
  • 我们可以根据不同手机屏幕的宽度，来动态设置 rem 的参考值：html 标签上的 font-size 的大小。
  • 用 lib-flexible 来设置根 font-size 的大小。

lib-flexible 引用方式

lib-flexible 对应的包名是 amfe-flexible，有两种引用方式：

npm 进行安装

  • npm i -S amfe-flexible

  • 然后在入口文件 main.js 中导入 amfe-flexible

直接导入压缩代码
